## Getting Started

Set up git hooks:

```sh
git config --local core.hooksPath .githooks
```

Run the development server:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Run the production build:

```sh
npm run build
```
